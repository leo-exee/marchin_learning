## Utilisation
1. Prétraitez les données :
    ```sh
    python src/data_preprocessing.py
    ```
2. Entraînez le modèle :
    ```sh
    python src/model_training.py
    ```
3. Évaluez le modèle :
    ```sh
    python src/model_evaluation.py
    ```
4. Prédisez de nouveaux commentaires :
    ```sh
    python src/predict.py
    ```
